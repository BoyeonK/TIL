## CLI 기초

Command Line Interface

GUI : Graphic User Interface



### 명령어

`pwd` 절대 경로 확인

`cd` 디렉토리 변경

- `cd 디렉토리이름` : 해당
- `cd ..` : 부모 디렉토리로 이동
- `cd -` : 이전 경로로 이동
- `cd ~` : ~경로로 이동

`ls` : 현재 경로의 파일/디렉토리 목록 확인

- `-a`: all 옵션
- `-l`: detail
- `al`: 같이도 가능!

`touch` : 파일 생성 (파일이름.확장자)

`mkdir` : 폴더 생성

`rm` : 파일 삭제

- `-rf` : 폴더 삭제

`mv` : 경로 이동 / 이름변경

- `mv 이동할파일 이동할경로`
- `mv 이름변경대상파일 변경할이름`

`start / open`

- 파일 / 폴더 열기
